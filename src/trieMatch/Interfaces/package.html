This package holds the interfaces that support match generation and other functions. The interfaces include
    1. A middle-layer wrapper for finding zeros, non-zeros and processing the instances of a group
      • QueryExpander<S> allocation and expansion of terms over a single data source
      • QueryBundleExpander<S>  allocation, expansion and combination of results for multiple data sources
    2. A pair of interfaces, QueryServer and QueryConnector, to be used with the QueryExpander
      • QueryServer<S>  "upper half" supports multplexing and caching
      • QueryConnector<S>  "lower half" gives conditional and unconditional probabilities, over groups and instances
    3. A functional predicate, foundP, typically "passed into" an evaluator; it is contained within
      • PredicateEvaluator<S> contains the foundP
      The Class ValidateCombination has processing logic, including decision rules tables, etc.
    4. An interface to combinatorial expansion (see class ChooseCombIF for lazy and eager iterators)
      • Chooser<S>
    5. Document and content iterators
      • ContentIterator<S>
       • DocIterator
